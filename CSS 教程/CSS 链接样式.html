四个链接状态是

  • a:link - 正常，未访问过的链接
  • a:visited - 用户已访问过的链接
  • a:hover - 当用户鼠标放在链接上时
  • a:active - 链接被点击的那一刻
快快快

当设置为若干链路状态的样式，也有一些顺序规则

  • a:hover 必须跟在 a:link 和 a:visited 后面
  • a:active 必须跟在 a:hover 后面
鼠标形状

default => 默认光标，箭头

pointer => 超链接的指针，手型

wait => 指示程序正在忙

help => 指示可用的帮忙

text => 指示文本

crosshair => 鼠标呈现十字状
