HTML 使用超级链接与网络上的另一个文档相连

HTML使用标签 <a>来设置超文本链接，是可以从一个页面指向另一个目的端的链接。
超链接可以是一个字，一个词，或者一组词，也可以是一幅图像，您可以点击这些内容来跳转到新的文档或者当前文档中的某个部分。

在标签 <a> 中使用了 href 属性来描述链接的地址

默认情况下，链接将以以下形式出现在
浏览器中:
  • 一个未访问过的链接显示为蓝色字体并带有下划线
  • 访问过的链接显示为紫色并带上下划线
  • 点击链接时，链接显示为红色并带上下划线
HTML 空链接

指指向链接后，鼠标变成手形，但单击后仍停留在当前页面

这是空链接 HTML链接-target属性

使用 Target 属性，你可以定义被链接的文档在何处显示（在新的窗口打开，还是在原有的窗口中打开）

默认的被链接文档会在原有的窗口中打开的。如果将 target 属性设置为 "_blank" 则文档就会在新窗口打开。

在新窗口打开百度 基本的注意事项 - 有用的提示 Note: 请始终将正斜杠添加到子文件夹。假如这样书写链接：href="//www.w3cschool.cn/html"，就会向服务器产生两次 HTTP 请求。这是因为服务器会添加正斜杠到这个地址，然后创建一个新的请求 HTML 链接 - id 属性

id 属性可用于在一个 HTML 文档中创建书签标记。

书签是不以任何特殊的方式显示，在 HTML 文档中是不显示的，所以对于读者来说是隐藏的
